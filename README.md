# dotfiles

[castaneai](https://github.com/castaneai)'s dotfiles


## Setup (common)

```sh
curl -sS https://starship.rs/install.sh | sh
```

## Setup (Arch Linux)

```sh
sudo pacman -S eza tig xdotool xorg-xhost
yay -S xkeysnail
./.xkeysnail/install.sh
```

## Setup (macOS)

```sh
brew install fish eza tig fzy
```
