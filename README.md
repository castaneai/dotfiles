# dotfiles

[castaneai](https://github.com/castaneai)'s dotfiles

## Setup (Ubuntu)

```sh
apt install python3-pip exa
pip3 install xkeysnail
git clone --depth 1 https://github.com/junegunn/fzf.git ~/.fzf
~/.fzf/install

./install.sh
```

## Setup (macOS)

```sh
brew install zplug exa
git clone --depth 1 https://github.com/junegunn/fzf.git ~/.fzf
~/.fzf/install

./install.sh
```
